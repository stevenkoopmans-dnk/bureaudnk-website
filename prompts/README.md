# Prompts

Templates voor terugkerende werkzaamheden. Plak de inhoud van een prompt direct in een Claude-sessie of laat Claude er actief naar verwijzen.

## Beschikbare templates

| Bestand | Wanneer gebruiken |
|---|---|
| [session-kickoff.md](session-kickoff.md) | Start van elke nieuwe sessie. Snelle context-pull. |
| [new-feature.md](new-feature.md) | Nieuwe feature of pagina-element bouwen. |
| [bug-fix.md](bug-fix.md) | Bug of regressie oplossen. |
| [copy-update.md](copy-update.md) | Alleen tekst aanpassen (geen layout/CSS). |
| [new-service-page.md](new-service-page.md) | Nieuwe dienstpagina aanmaken vanuit `dienst-template.html`. |
| [deploy-check.md](deploy-check.md) | Verificatie na deploy (productie-test). |

## Conventies

- Templates zijn in NL, passend bij de tone
- Iedere template start met "Lees eerst" en verwijst naar de relevante docs
- Templates zijn instructies voor Claude, geen brief aan Steve
